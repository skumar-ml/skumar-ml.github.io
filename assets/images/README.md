# Images Directory

This directory contains images used throughout the website.

## Important Files

- `profile.jpg` - Your profile picture that appears on the home page and about page
- `project1.jpg`, `project2.jpg`, etc. - Images for your research projects
- Other images used in blog posts or other pages

## Image Guidelines

- For profile pictures, a square image with dimensions of at least 500x500 pixels is recommended
- For project images, use images with a 16:9 or 4:3 aspect ratio
- Optimize images for web (compress them) to ensure fast loading times
- Use descriptive filenames

## Adding Your Profile Picture

1. Replace the `profile.jpg` file with your own photo
2. Make sure it's named `profile.jpg` (or update the references in the code if using a different filename)
3. If you don't have a photo ready, you can use a placeholder image temporarily
